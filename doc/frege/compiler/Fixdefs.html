Compiled: Thu Oct 20 00:58:07 CEST 2011 from source file: .\frege\compiler\Fixdefs.fr

Package Documentation

This package implements the first compiler pass after lexical analysis. It fixes the list of definitions by incorporating document comments into definitions, and joining separate equations of functions that come out as separate D.DefinitionT.FunDcl definitions from parsing.

The function fixdefs will also be used from other modules for fixing sub definitions.
