pass :: Data.StIO Data.Global (String, Int)

In this pass, we insert placeholders in the symbol table and create source code for derived instances.

Instance declarations come later in pass2, because references to classes and data types must be resolved. Likewise derive declarations, they are transformed to instance declarations.

pass2 :: Data.StIO Data.Global (String, Int)

pass2 creates the field definitions.

requires: data types and constructors are entered

ensures: for each type T, for each field f in T there wil be 4 corresponding functions T.f, T.upd$f, T.chg$f and T.has$f
