simplify a Data.Rho Data.Context

  • if context is of the form C a or C (a b ...), check that it is not implied by other contexts, i.e. (Ord a, Num a) is simplified to Num a
  • if context is of the form C (T ...) make sure that instance exists and add its implications, i.e. Eq [a] will add Eq a.
