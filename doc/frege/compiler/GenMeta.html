Package frege.compiler.GenMeta

Compiled: Thu Oct 20 00:59:39 CEST 2011 from source file: .\frege\compiler\GenMeta.fr

Package Documentation

Generate interface data for frege packages.

The interface data are custom java annotations on the class that is generated from a package. The format is definded in frege/MD.java.

The information will be used by the compiler, the documentation tool and the quickcheck tool.
